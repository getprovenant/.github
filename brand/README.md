# Provenant brand

The Provenant mark is a **lineage graph**: three nodes branching up from a single amber
origin — provenance traced back to its source. The amber origin node also nods to Rust.

## Colors
- Slate ground — `#171B24`
- Mark (off-white) — `#F1ECE4`
- Origin / accent (amber) — `#E08A3C`

## Files
- `provenant-mark.svg` — source (120×120 viewBox, colors inlined)
- `provenant-avatar-512.png` — 512×512 org profile picture
